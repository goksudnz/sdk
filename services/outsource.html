    OutsourceProcess Servis Dokümantasyonu

    Bu doküman, InsurGateway'in OutsourceProcess servisinde kullanabileceğiniz metotları barındırır. Servisin doğru kullanımı için gerekli olan bilgileri metotlara ait açıklamalar içerisinde bulabilirsiniz.

    GetOutsourceProcess

    InsurGateway üzerinden sigorta şirketlerinin sağladığı kaynak servislerine erişebilmenizi sağlayan servistir. Bu servisi kullanarak riziko adres sorgulama, müşteri sorgulama, vb. şirket servisleriyle haberleşebilirsiniz.

    Parametreler

    İlgili metodun ihtiyaç duyduğu parametreler ve bunların açıklaması aşağıda tablo halinde verilmiştir.

    İsimTipAçıklama
    outsourceProcessIdintServiste kullanılacak Outsource Servis tipine ait id bilgisidir.
    keyValuePairsDictionary<int, List<string>>Servise gönderilecek key - value değerlerini tutar. İhtiyaç halinde key için birden fazla değer gönderilebilir.

    keyValuePairs parametresi boş gönderildiği taktirde, SDK tarafından boş gönderilemeyeceğine yönelik hata mesajı dönecektir.

    ⚠️ keyValuePair için null veya boş değer olamaz!

    Zorunlu Keyler

    Servisin geliştirme kiti tarafından başarılı şekilde InsurGateway servislerine erişebilmesi için metot özelinde zorunlu tutulan keyler aşağıda belirtilmiştir.

    Belirtilen keyler gönderilmediği taktirde SDK, InsurGateway servislerini tetiklemez.

    113Acente Id
    114Acente Kullanıcı Adı
    115Acente Şifresi

    Cevap Nesnesi

    Servisten dönecek olan alanlar ve bunlara ait açıklamalar aşağıda tablo halinde verilmiştir.

    İsimTipAçıklama
    ResponseKeyValuesList<KeyValueSet>Key - Value olacak şekilde değer seti döner.
    ParameterValuesList<ParameterValueSet>Parametrik değer listesi döner.

    Kullanım Örneği

    // kullanılacak outsource process id'si belirlenir.
var outsourceProcessId = 1;

// gönderilecek key-value listesini oluşturur.
var keyValueList = new Dictionary<int, List<string>>()
{
    { 113, new List<string>() { "1" }},
    { 114, new List<string>() { "username" }},
    { 115, new List<string>() { "password" }},

    { 178, new List<string>() { "34" } }
};

// asenkron bir metot içerisindeysek:
var response = await new InsurGatewaySDK.Shared.OutsourceProcess()
                            .GetOutsourceProcess(outsourceProcessId, keyValueList);

// write your codes

    Eğer asenkron bir metot içerisinde değilsek ve await operatörünü kullanamıyorsak, response aşağıdaki şekilde de alınabilir.

    var response = Task.Run(async () => await new InsurGatewaySDK.Shared.OutsourceProcess().GetOutsourceProcess(outsourceProcessId, keyValueList))
                    .ConfigureAwait(false)
                    .GetAwaiter()
                    .GetResult();
