    Doküman Servis Dokümantasyonu

    Bu doküman, InsurGateway'in Doküman servisinde kullanabileceğiniz metotları barındırır. Servisin doğru kullanımı için gerekli olan bilgileri metotlara ait açıklamalar içerisinde bulabilirsiniz.

    GetDocument

    InsurGateway üzerinden sigorta şirketlerinden doküman sorgulayabileceğiniz servistir. Bu servisi kullanarak sigorta şirketinden ilgili teklif, poliçe, bilgilendirme vb. dokümanınlarını sorgulayabilirsiniz.

    Parametreler

    İlgili metodun ihtiyaç duyduğu parametreler ve bunların açıklaması aşağıda tablo halinde verilmiştir.

    İsimTipAçıklama
    keyValuePairsDictionary<int, List<string>>Servise gönderilecek key - value değerlerini tutar. İhtiyaç halinde key için birden fazla değer gönderilebilir.

    keyValuePairs parametresi boş gönderildiği taktirde, SDK tarafından boş gönderilemeyeceğine yönelik hata mesajı dönecektir.

    ⚠️ keyValuePair için null veya boş değer olamaz!

    Zorunlu Keyler

    Servisin geliştirme kiti tarafından başarılı şekilde InsurGateway servislerine erişebilmesi için metot özelinde zorunlu tutulan keyler aşağıda belirtilmiştir.

    Belirtilen keyler gönderilmediği taktirde SDK, InsurGateway servislerini tetiklemez.

    KeyAçıklama
    113Acente Id
    114Acente Kullanıcı Adı
    115Acente Şifresi

    ⚠️ Acente giriş bilgileri eksik!

    KeyAçıklama
    68InsurGateway Teklif Numarası
    127InsurGateway Ürün Kodu
    285Doküman Tipi

    ⚠️ Zorunlu keyler eksik!

    Key Parametreleri: 285

    285 numaralı key, doküman tiplerini simgeler. Bu keye ait gönderilebilecek parametreler aşağıda tablo halinde verilmiştir.

    DeğerAçıklama
    1Poliçe Pdf
    2Teklif Pdf
    3Bilgilendirme Formu
    4Zeyil Pdf
    5Makbuz Pdf
    6KVKK Onay Formu
    7E-Ticaret Onay Formu

    Bu parametrelerin dışında bir değer gönderiminde, InsurGateway hatalı parametrik değer hatası dönecektir.

    ⚠️ (285 - Döküman Tipi), nolu key'ler için girilen değerler parametre listelerinde mevcut değil.

    Cevap Nesnesi

    Servisten dönecek olan alanlar ve bunlara ait açıklamalar aşağıda tablo halinde verilmiştir.

    İsimTipAçıklama
    DocumentUrlstringDokümana ait url bilgisini barındırır.
    Documentbyte[]Byte dizisi olarak doküman içeriğini barındırır.

    Kullanım Örneği

    // gönderilecek key-value listesi oluşturulur.
var keyValueList = new Dictionary<int, List<string>>() 
{
    { 113, new List<string>() { "1" }},
    { 114, new List<string>() { "username" }},
    { 115, new List<string>() { "password" }},

    { 68, new List<string>() { "3" }},
    { 127, new List<string>() { "2178" }},
    { 285, new List<string>() { "3" }}
};

// asenkron bir metot içerisindeysek:
var response = await new InsurGatewaySDK.Shared.Document()
                            .GetDocument(keyValueList);

// write your codes

    Eğer asenkron bir metot içerisinde değilsek ve await operatörünü kullanamıyorsak, response aşağıdaki şekilde de alınabilir.

    // senkron bir metot içerisindeysek:
var response = Task.Run(async () => await new InsurGatewaySDK.Shared.Document().GetDocument(keyValueList))
                   .ConfigureAwait(false)
                   .GetAwaiter()
                   .GetResult();

// write your codes
